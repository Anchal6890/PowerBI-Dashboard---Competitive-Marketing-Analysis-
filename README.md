# Job Market Analytics Dashboard

**Tools:** Power BI (Microsoft Fabric), Power Query, DAX, Excel


## Project Overview
This project is an **interactive Power BI dashboard** designed to analyze the **job market for Data Analyst roles**, focusing on trends across job roles, locations, experience levels, and skill requirements. The dashboard provides insights into job availability, salary distribution, and experience demand, helping both recruiters and job seekers make informed decisions.

The dashboard was created in **Microsoft Fabric (Power BI)**, with KPIs, calculated measures, and interactive visuals to support career and hiring analysis.

---

## Features
- **Interactive Dashboard:** Explore job market trends by role, location, and experience level.  
- **KPIs & Calculations:** Track total jobs, average salary, and experience demand using DAX.  
- **Dynamic Filters & Slicers:** Analyze data by job roles, locations, and required skills.  
- **Business Insights:** Visuals and charts designed for clear interpretation by non-technical users.  
- **Dashboard Design:** Professional layout showcasing real-world data analytics skills.

## How to Use
1. **Download the `.pbix` file** from this repository.  
2. Open the file in **Power BI Desktop (Fabric version recommended)**.  
3. Note: Since the dashboard uses a live dataset in Microsoft Fabric, some visuals may not display actual data if opened offline.  
4. Explore the dashboard structure, filters, KPIs, and measures to understand the analysis.
